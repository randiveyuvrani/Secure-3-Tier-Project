# ✅ 3-Tier Web Architecture on AWS Using Terraform

## 🔧 Components
- VPC with public/private subnets across 2 AZs
- EC2 Web + App Tiers
- RDS MySQL DB Tier
- Modular Terraform
- Shell-based provisioning

## 🚀 Deploy Instructions

```bash
terraform init
terraform apply -auto-approve
```
